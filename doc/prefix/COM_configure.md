# USB Serial Port(COM3) 屋性

## 端口设置

- 每秒位数(B): 9600
- 数据位(D)  : 8
- 奇偶校验(P): 无
- 停止位(S)  : 1
- 流控制(F)  : 无

## COM3 的高级设置

- COM 端口号(P): COM3
- USB传输大小 
    - 接收(字节): 4096
    - 发送(字节): 4096
- BM选项
    - **延迟计时器(毫秒): 2**
- 超时
    - 最小读超时(毫秒): 0
    - 最小写超时(毫秒): 0
- 其他选项
    - [x] 串行枚举器
    - [ ] 串行打印机
    - [ ] 如果电源关闭则取消 
    - [x] **意外移除时事件**
    - [ ] 关闭时设置RTS
    - [ ] 启接收(字节):Modem Ctrl
    - [ ] Enable Selective Suspend
    - Selective Suspend Idle Timeout (secs): 5
